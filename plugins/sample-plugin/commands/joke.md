---
name: joke
description: Tells a joke based on a topic chosen by the user
---

Tell a funny, clean joke about the topic provided by the user. If no topic is provided, ask the user what topic they'd like a joke about.

The joke should be:
- Family-friendly and appropriate for all audiences
- Related to the specified topic
- Original and creative
- Well-structured with a clear setup and punchline

Format the response as just the joke itself without additional commentary.
